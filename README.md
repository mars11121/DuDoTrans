# DuDoTrans

- This code is an official implementation of "DuDoTrans: Dual-Domain Transformer for Sparse-View CT Reconstruction" based on the open source ct reconstruction toolbox [odl](https://github.com/odlgroup/odl).

## Installation

### Requirements

- platform: linux-64
- python=3.6.13
- pytorch=1.8.1
- odl=1.0.0
- scikit-learn=0.24.2
- scikit-image=0.17.2
- scipy=1.5.4
- CUDA 11.0 or higher
